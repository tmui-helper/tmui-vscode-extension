# tmui-helper for vscode
此插件希望能帮助开发者提高使用 TMUI3.0组件库 开发项目的效率。

## 特色
  + 鼠标悬浮查看组件文档
  + 代码自动补全
  + 组件内键入空格自动提示组件属性列表（暂不支持公共属性和选中属性自动补全属性可选项）

## 用法
### 1、移动鼠标到 TMUI3.0 组件标签上即可在悬浮提示框架下面看到对应组件的文档信息。
